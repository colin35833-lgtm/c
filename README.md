## 这是什么？

一个**单 HTML 文件**的个人工作台，集成了：
- ✅ 日常任务管理（带拖拽排序）
- ✅ 利润追踪（当日利润 + 累计总利润，自动归档）
- ✅ 主线/可选任务（次日自动归档）
- ✅ 链接收藏（待开发列表）
- ✅ 云端备份（手动同步到 Supabase）

## 核心功能

### 📋 日常任务
- 层级结构：分类（parent）→ 子任务（child）
- 子任务填写利润（¥）后自动标记完成
- 利润支持负数输入（退款扣减）
- parent/child 均可设置链接（🔗），点击 ↗ 直接跳转
- 双击任务名可改名，自动保存

### 💰 利润追踪
- 子任务利润实时汇总到顶部
- 每日重置时自动归档到收支明细
- 总利润跨天累加
- 收支明细按天查看，一行展示

### 🎯 主线/可选任务
- 每天自动归档已完成任务
- 支持拖拽排序
- 也支持设置链接跳转

### 🔒 数据安全
- 锁屏密码保护
- 数据加密后上传云端（XOR + Base64）
- 手动同步（不自动），杜绝数据混乱
- 首次同步 5 秒超时保护

## 使用说明

### 第一次使用
1. 打开 https://colincolin.cc.cd
2. 设置锁屏密码
3. 开始使用

### 日常操作
- **添加任务**：点击分类旁的「+」
- **完成任务**：点击任务前的圆圈 ✓
- **记录利润**：在子任务的 ¥ 输入框输入金额
- **设置链接**：点击子任务旁的 🔗，输入 URL
- **拖拽排序**：直接拖动任务卡片

### 数据同步
- ☁ 上传：把本地数据推送到云端
- ⬇ 下载：从云端拉取数据覆盖本地

## 技术架构

```
┌──────────────┐     push      ┌─────────────┐
│  index.html  │ ──────────→   │   GitHub    │
│  (单文件)    │    (git)      │  (源码仓库)  │
└──────────────┘               └──────┬──────┘
       │                              │
       │  Cloudflare Pages             │ auto build
       ▼                              ▼
┌──────────────┐               ┌─────────────┐
│  colincolin  │               │   colin-    │
│  .cc.cd      │               │ homepage    │
│  (生产环境)   │               │  .pages.dev │
└──────────────┘               └─────────────┘
       │
       │ 手动 sync/down
       ▼
┌──────────────┐
│  Supabase    │
│  (云端备份)   │
└──────────────┘
```

- 数据存 localStorage（本地主存储）
- Supabase 仅作云端备份（手动操作）
- 无数据库、无后端、无框架
